- 基于链表的 Partition 操作，非常有意思！！！

我写的题解：[https://leetcode-cn.com/problems/partition-list-lcci/solution/bu-gai-bian-lian-biao-jie-dian-nextzhi-j-ll1f/](<https://leetcode-cn.com/problems/partition-list-lcci/solution/bu-gai-bian-lian-biao-jie-dian-nextzhi-j-ll1f/>)

```c++
/**
 * Definition for singly-linked list.
 * struct ListNode {
 *     int val;
 *     ListNode *next;
 *     ListNode(int x) : val(x), next(NULL) {}
 * };
 */
 // 就是把比 x 小的数放到一起，整体在大于等于 x 的数之前就行了！
class Solution {
public:
    ListNode* partition(ListNode* head, int x) {
      ListNode* fast = head;
      ListNode* slow = head;
      while (fast) {
        while (fast->val >= x && fast->next)
          fast = fast->next;
        swap(fast->val, slow->val);
        fast = fast->next;
        slow = slow->next;
      }
      return head;
    }
};
```

[面试题 05.02. 二进制数转字符串](https://leetcode-cn.com/problems/bianry-number-to-string-lcci/)

- 小数的移位是怎么样的？

- 我竟然想到计算机中的浮点数相关的知识了，很多都忘了。

```c++
class Solution {
public:
    string printBin(double num) {
      string res = "0.";
      double k = 0.5;
      // sizeof(double) == 8
      while (res.size() <= 32 && num != 0) {
        if (num >= k) {
          res += '1';
          num -= k;
        }
        else {
          res += '0';
        }

        k /= 2; // 1/2, 1/4, 1/8, 1/16, 1/32, 1/64, ...
      }
      return num==0 ? res : "ERROR";
    }
};
```
