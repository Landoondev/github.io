[面试题 08.05. 递归乘法](https://leetcode-cn.com/problems/recursive-mulitply-lcci/)

有一点俄式乘法的意思。

- 溢出？
- 负数的情况？先当做是整数，最后结果加上负号。

```C++
class Solution {
public:
    int multiply(int A, int B) {
      if (A == 0 || B == 0)
        return 0;
      
      return _multiply(A, B);
    }

 private:
    int _multiply(int A, int B) {
      if (!(B >> 1))
        return A;
      if (B & 1) {
        return A + _multiply(A << 1, B >> 1);
      }
      else {
        return _multiply(A << 1, B >> 1);
      }
    }
};
```

评论区发现另外一个解法，非常简洁。

乘法的本质。

```c++
class Solution {
public:
    int multiply(int A, int B) {
      return (B | 0) ? A + multiply(A, B - 1) : 0;
    }
};
```
